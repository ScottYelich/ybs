# Framework Research

This directory contains research documents that informed the design and evolution of the YBS framework.

## Documents

### spec-system-research.md (2026-01-17)

Comprehensive research on handling cross-cutting concerns (accessibility, UX, UI, error handling, i18n) and BASE specification patterns.

**Topics covered:**
- Where cross-cutting concerns belong in existing spec types
- BASE spec system with inheritance ($ref pattern)
- Context-driven spec requirements (not all specs needed for all features)
- Industry research: W3C Design Tokens, WCAG 2.1, design systems

**Result**: Led to implementation of BASE spec templates and cross-cutting concerns documentation in framework/methodology/writing-specs.md
